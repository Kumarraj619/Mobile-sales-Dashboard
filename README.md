# 📱 Mobile Sales Insights Dashboard  

An interactive **Power BI dashboard** designed to analyze mobile phone sales performance across brands, cities, and customer segments.  
*Built while learning from Satish Dhawale Sir’s YouTube Power BI tutorials.*

---

## 📌 Short Description  

The **Mobile Sales Insights Dashboard** provides a complete overview of mobile sales data, highlighting revenue, quantity sold, transactions, pricing trends, customer ratings, and payment methods.

It helps businesses and analysts quickly understand brand performance, regional trends, and customer behavior for better decision-making.

---

## 🛠️ Tech Stack  

- 📊 **Power BI Desktop** – Dashboard creation and visualization  
- 🔄 **Power Query** – Data cleaning and transformation  
- 🧠 **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- 🔗 **Data Modeling** – Relationships for cross-filtering and aggregation  
- 📁 File Format – `.pbix`  

---

## 📂 Data Source  

Retail mobile sales dataset containing:

- Brand-wise sales (Apple, Samsung, OnePlus, Vivo, Xiaomi)  
- Monthly sales data (January–December)  
- City-wise sales distribution  
- Customer ratings (1–5 scale)  
- Payment methods (UPI, Debit Card, Credit Card, Cash)  
- Quantity sold and transaction details  

---

## 🚀 Features & Highlights  

### 🔎 Business Problem  

Mobile retailers manage large volumes of sales data across multiple brands and regions. Raw data makes it difficult to quickly identify top-performing products, cities, and sales trends.

---

### 🎯 Goal of the Dashboard  

To build an interactive analytical tool that:

- Tracks overall sales performance  
- Enables brand, city, and month-level analysis  
- Analyzes customer ratings and payment behavior  
- Supports data-driven business decisions
